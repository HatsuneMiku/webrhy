
       Webrhy - Web browser based eazy beat machine

1. DESCRIPTION
  Webrhy is a beat machine for real time performance.
  It is all written in JavaScript using Web Audio API
  and Web MIDI API technology.

2. FEATURES
  - 2 types of instuments (External MIDI device/Internal Synth)
  - 8 patterns x 3 tracks (Synth/Bass/Drum)
  - Automatic random beat pattern generator
